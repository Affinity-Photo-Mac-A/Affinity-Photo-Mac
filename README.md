## Overview

<a href="#">Affinity Photo 2 for Mac</a> expands its professional toolkit with advanced liquify, warp, and mesh distortion tools that give retouchers and compositors precise geometric control over image content, alongside a batch processing engine and macro recording system that automate repetitive editing workflows across large image collections. These capabilities address the full production workflow of professional photographers — from individual image retouching through efficient batch delivery.

The <a href="#">Liquify Persona</a> provides a dedicated workspace for organic image deformation with mesh-based warp, push, pinch, twirl, and reconstruct tools operating on a live preview of the image. Portrait retouching, fashion editing, and creative distortion work all benefit from the precision of the Liquify Persona's mesh visualization, which makes the deformation geometry visible and controllable rather than applying distortion as a black-box operation. <a href="#">Mesh warp tool</a> places a customizable control grid over any image region for structured distortion that follows a regular pattern — perspective corrections, architectural straightening, and product shape adjustments all map naturally to mesh-based control rather than freeform brush deformation.

<a href="#">Macros recording system</a> captures sequences of editing operations as replayable automation scripts. A retouching workflow applied to a portrait — frequency separation setup, skin tone adjustment, sharpening, and export preparation — records once and replays on any number of additional images with a single click. <a href="#">Batch processing</a> applies saved macros across entire folders of images, delivering a consistent editing treatment to hundreds of files without opening each one individually. Wedding photographers processing full event galleries, product photographers editing entire catalog shoots, and real estate photographers delivering multiple property shoots all benefit from the dramatic time reduction that macro-based batch processing provides.

<a href="#">360-degree equirectangular image editing</a> handles the spherical panoramas produced by 360-degree cameras with a dedicated spherical projection mode that allows corrections and retouching to be applied in the equirectangular format and previewed in an interactive spherical view. <a href="#">Export Persona</a> configures multi-format simultaneous export — generating web-optimized JPEG, full-resolution TIFF, and print-ready PDF versions of a finished image in a single export operation — with slice tools that export different regions of a document to separate files for web and UI asset production.

- <a href="#">Batch processing folder automation</a> applies saved macros to entire image folders delivering consistent editing without opening each file individually
